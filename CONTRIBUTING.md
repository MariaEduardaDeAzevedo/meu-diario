## Quero contribuir com funcionalidades!

Para contribuir com funcionalidades, você pode resolver uma de nossas Issues em aberto (seja USs ou outras categorias) ou implementar funcionalidades que deseje! Basta [abrir uma PR](#como-criar-uma-pr) com sua adição de código!

## Quero contribuir com ideias!

Se você tem uma ideia, mas não se sente confortável ainda para contribuir com código ou não está com muito tempo, você pode contribuir com sua ideia e deixar a possibilidade de implementação para depois ou para a comunidade! Basta que:

1. Crie uma Issue na aba de issues do repositório, descrevendo sua funcionalidade. Não esqueça de indicar o ID da US (observe as Issues já criadas anteriormente para tomar como base).
2. Adicione a sua US no README.md do projeto. Basta que você edite o README.md adicionado a sua US na listagem de USs do sistema e [abrir uma PR](#como-criar-uma-pr).  

## Como criar uma PR?

1. Faça um fork deste repositório para sua conta do GitHub.
2. Clone o repositório referente ao fork que você fez anteriormente.
3. Realize suas modificações na branch main do seu repositório. Faça quantos commits e pushes achar necessário.
4. Crie uma PR do seu fork para o repositório principal! Você pode fazer isso utilizando a própria interface do GitHub.

> Consulte a documentação do GitHub [neste link](https://docs.github.com/pt/get-started/quickstart/fork-a-repo) para aprender a realizar este passo a passo! Para o passo de abrir uma PR você pode consultar [este link](https://docs.github.com/pt/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/about-pull-requests).
